*Todoster*

Todoster is an simple Todo list built using Test Driven Development with technologies with Ruby on Rails in the backend and Javascript and jQuery in the front end.

Todoster can be found in production [here](https://todoster-ethan-he.herokuapp.com/).

**Start Application**

1. Change directory to Todoster `cd todoster`
2. Start server `rails server`
